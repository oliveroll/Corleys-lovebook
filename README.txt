# Corley Lovebook (Offline)

## Run
Open `index.html` in a browser (Chrome, Safari, Firefox). Works fully offline.

## Add your photos
Put images in `photos/`:
- cover.jpg (required)
- p1.jpg, p2.jpg, ... (match names in app.js)

## Customize text
Open `app.js` and edit the `CONTENT` object (intro letter, timeline, notes, captions).
